***Αρχική σελίδα: 
Κάποιες σύντομες πληροφορίες για τι τι μπορεί να κάνει η κάθε ομάδα χρηστών δίνονται στη κεντρική σελίδα, σε μορφή carousel.
Στο header Υπάρχει καρτέλα με τα Μαθήματα, όπου μπορεί κάποιος να πάρει πληροφορίες για κάθε μάθημα του τμήματος.
Στα δεξιά υπάρχει η επιλογή της σύνδεσης, όπου κάποιος μπορεί είτε να κάνει είσοδο με τον λογαριασμό του, είτε να δημιουργήσει νέο.

***Φοιτητές/τριες:
Οταν κάποιος κάνει σύνδεση, στο header εμφανίζονται άλλες δύο καρτέλες (Φοιτητές - Καθηγητές). Ανάλογα με τον ρόλο του κάθε χρήστη 
οι καρτέλες θα είναι ενεργές ή ανενεργές. Οι επιλογές του Φοιτητή είναι: 
Δηλώσεις - Βαθμολογίες- Πιστοποιητικά - Βοήθεια
Δηλώσεις: 
Στη σελίδα αυτή ο χρήστης έχει την επιλογή να ξεκινήσει μια νέα δήλωση, ή να δει το ιστορικό προηγούμενων δηλώσεών του.
Στη νέα δήλωση επιλέγει τα μαθήματα τα οποία είναι χωρισμένα ανά εξάμηνο, υπάρχει έλεγχος στον αριθμό των μαθημάτων, και μόνο όταν 
έχει επιλέξει κάποιο μάθημα - και λιγότερα από το ανώτερο όριο του έτους του - μπορεί να προχωρήσει
Στη συνέχεια υπάρχει μια προεπισκόπηση των επιλογών του. Μπορεί να γυρίσει πίσω και να τις επεξεργαστεί, ή να τις αποθηκεύσει προσωρινά, 
ή να επιλέξει υποβολή. Εάν μια δήλωση έχει αποθηκευτεί προσωρινά, μπορεί ο χρήστης να την βρει στο ιστορικό των δηλώσεων, να την επεξεργαστεί 
για να την υποβάλει, ή να την διαγράψει τελείως.

Βαθμολογίες: 
Εδώ εμφανίζονται όλες οι βαθμολογίες του φοιτητή, είτε συγκεντρωτικά, είτε χωρισμένες σε επιτυχείς και ανεπιτυχείς, είτε χωρισμένες ανά εξεταστική περίοδο.
Δίνεται η επιλογή να κάνει εξαγωγή σε pdf ή απευθείας εκτύπωση.

Πιστοποιητικά:
Οι επιλογές είναι αντίστοιχες με της Δήλωσης (Νέα αίτηση - ιστορικό πιστοποιητικών). Κατά τη διαδικασία της αίτησης ο χρήστης μπορεί να επιλέξει ένα 
πιστοποιητικό, ενώ κάνοντας κλικ σε ένα πιστοποιητικό διαβάζει μια σύντομη περιγραφή. Επιλέγοντας 'Επόμενο' κατευθύνεται σε μια προεπισκόπηση, όπου βλέπει 
τα στοιχεία του προσυμπληρωμένα, πριν την υποβάλει οριστικά. 
Στο ιστορικό των πιστοποιητικών ο χρήστης μπορεί να δει το είδος των πιστοποιητικών, την ημερομηνία και ώρα έκδοσής τους, και του δίνονται κάποιες επιλογές για 
προβολή, εκτύπωση και αποθήκευση των εγγράφων.

Βοήθεια:
Δίνονται ενδεικτικά απαντήσεις και σύντομες οδηγίες σε ερωτήματα που μπορεί να έχει ένας χρήστης.

***Καθηγητές/τριες:

Οι επιλογές που υπάρχουν στην καρτέλα τους είναι: Διαχείριση Μαθημάτων, Βαθμολόγια, Βοήθεια

Διαχείριση Μαθημάτων:
Φαίνονται τα μαθήματα στα οποία μπορεί ένας καθηγητής να βάλει βαθμούς, το πλήθος των εγγεγραμμένων φοιτητών, και επιλογές για να 
μεταφερθεί στο eclass του μαθήματος, ή στη δημιουργία νέου βαθμολογίου.

Βαθμολόγια: 
Πρώτα επιλέγεται ένα μάθημα από τα μαθήματα στα οποία μπορεί να βάλει βαθμολογίες. Ολοι οι φοιτητές εμφανίζονται
στη λίστα, και για τον καθένα πρέπει να μπει ένας βαθμός. Για να προχωρήσει στο επόμενο βήμα, γίνεται έλεγχος ορθότητας 
(να έχουν μπει βαθμοί σε όλους, και μεταξύ 0 και 10.) Δίνεται ακόμη η επιλογή μεταφόρτωσης εγγράφου για να εισαχθούν οι βαθμολογίες. 
(TIP: επιλέγουμε εισαγωγή από αρχείο για να μπουν τυχαίοι βαθμοί σε όλους και να προχωρήσουμε στο επόμενο βήμα γρήγορα).
Τα επόμενα βήματα (Προεπισκόπηση - Υποβολή) ακολουθούν τη λογική των δηλώσεων.
Δίνεται και εδώ η δυνατότητα προσωρινής αποθήκευσης, και επεξεργασίας σε ύστερο χρόνο.

Βοήθεια: Επίσης στην ίδια λογική με των φοιτητών.
